# Token faucet bot example using the Sonic testnet.

## Installation

```bash
npm install
```

## Add data to .env file

```bash
WALLET_PRIVAT_KEY= 'your private wallet key'
WALLET_ADDRESS= 'your wallet address'
FAUCET_URL= 'https://norma-p.testnet.soniclabs.com'
```

## Running the bot

```bash
$ npm start
```
